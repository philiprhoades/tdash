tDash Image upload API

OAuth based API

tDash supports authenticating with OAuth credentials.

We will support OAuth delegation as soon as it matures. Until then, your app will need to share its OAuth credentials with us the traditional way. We don't store any of your credentials on our servers. We just use them for account verification and then dispose them immediately.

URL

http://tdash.org/pic/uploadOAuth.format

Where format is xml or json

HTTP Method

POST

Rate Limit

The API is not rate limited. However there is a global limit on the number of images a user can upload per day (across all interfaces including the API)

Parameters

Note that the request has to encoded as a multi-part form/data.

NameRequired?Descr
imageYesThis is the content of the image (GIF, JPEG, PNG)
descrNoDescription to go along with the image. This will be used to compose the tweet.
oauth_consumer_keyYesYour app's Oauth consumer key
oauth_consumer_secretYesYour app's Oauth consumer secret
oauth_tokenYesYour app user's Oauth token
oauth_token_secretYesYour app user's Oauth token secret
